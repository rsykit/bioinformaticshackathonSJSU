# BioQuest

An interactive bioinformatics learning tool built for the SJSU Hackathon (Track 1). Upload a research paper and BioQuest generates a full visual lab experience powered by OpenAI.

## Features

- **Paper Upload** — paste or upload any bioinformatics paper to generate content
- **DNA Animation** — live double helix visualization
- **Pathway Map** — auto-generated biological pathway diagram from your paper
- **Data Charts** — key quantitative findings visualized as bar charts
- **Key Concepts** — glossary of terms extracted from the paper
- **AI Chat** — ask questions about your paper with a bioinformatics tutor
- **Visual Quiz** — sequence, table, and diagram challenges with video hints

## Setup

1. Get an [OpenAI API key](https://platform.openai.com/api-keys) (requires `sk-...` key)
2. Open `bioquestt.html` in your browser
3. Enter your API key in the top-right field
4. Upload or paste a bioinformatics paper and click **Analyze**

## Tech Stack

- Vanilla HTML/CSS/JavaScript (single file, no dependencies)
- OpenAI GPT-4o API for content generation and chat
- Canvas API for DNA and chart visualizations
